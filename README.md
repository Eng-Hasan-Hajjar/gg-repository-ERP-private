## 📋 نظرة عامة

**Namaa Academic ERP** هو نظام إدارة موارد مؤسسية (Enterprise Resource Planning) شامل ومتقدم، تم تطويره من الصفر لخدمة مؤسسة **نماء أكاديمي** — المؤسسة التعليمية الأولى على مستوى الشرق الأوسط المتخصصة في تقديم الدبلومات المهنية والدورات التدريبية الاحترافية عبر فروعها في تركيا وألمانيا وسوريا.

يغطي النظام **جميع العمليات المؤسسية** من إدارة الطلاب والموظفين والمالية والحضور والامتحانات والمبيعات (CRM) والأصول والبرامج التعليمية وصولاً إلى التقارير التنفيذية المتقدمة — في منصة واحدة متكاملة وآمنة.

## 🎯 الوحدات الرئيسية (Core Modules)

### 1. 📊 لوحة التحكم التنفيذية (Executive Dashboard)

- لوحة تحكم ذكية مع إحصائيات حية لجميع الأقسام
- 4 مؤشرات أداء سريعة (KPIs): تنبيهات، مالية، طلاب، نشاط النظام
- كروت إحصائية مدمجة داخل كل وحدة مع أرقام محدّثة لحظياً
- تقسيم ذكي للصفحة: عمليات أساسية → موارد بشرية → بنية تحتية

### 2. 🎓 إدارة الطلاب (Student Management)

- إدارة شاملة لملفات الطلاب مع حالات التسجيل (معلّق/مثبّت/مؤجل)
- ربط الطلاب بالدبلومات والفروع مع نظام أقساط ودفعات
- ملفات طلاب موسعة مع بيانات إضافية وشهادات ومرفقات
- نظام تأكيد التسجيل مع Timestamps

### 3. 📞 إدارة علاقات العملاء — CRM

- نظام Leads متكامل مع مراحل المتابعة (جديد → متابعة → مهتم → مسجل)
- نظام اختيار دبلومات ذكي مع بحث فوري وفلترة حسب الفرع والرمز
- ربط العملاء بالدبلومات المتعددة مع اختيار الرمز والفرع لكل دبلومة
- سجل متابعات مفصل وتقارير تحويل ومصادر العملاء
- تحويل تلقائي من عميل محتمل إلى طالب مسجل

### 4. 💰 النظام المالي (Financial System)

- صناديق مالية متعددة مع دعم العملات المختلفة
- حركات مالية (مقبوض/مدفوع/مناقلة) مع نظام ترحيل
- ربط الحركات بالحسابات المالية (Polymorphic: طلاب/عملاء)
- تقارير مالية يومية وربحية حسب البرنامج
- نظام أقساط طلاب مع خطط دفع مرنة

### 5. 👥 الموارد البشرية (HR Module)

- إدارة موظفين ومدربين مع ملفات شاملة
- نظام عقود مع تواريخ بداية/نهاية وتجديد
- مستحقات مالية مع ربط بالصناديق وحالة الدفع
- ربط المدربين بالدبلومات والفروع
- إحصائيات مدمجة: عدد المدربين/الموظفين النشطين

### 6. ⏰ نظام الحضور والدوام (Attendance System)

- تسجيل دخول/خروج مع احتساب تلقائي للتأخير والساعات
- **نظام استراحة ذكي**: بدء/إنهاء الاستراحة مع حساب صافي الساعات
- توليد تلقائي لسجلات الأسبوع من جداول الشيفتات
- تقويم شهري مرئي مع تصدير Excel/PDF
- أزرار ذكية تظهر/تختفي حسب حالة السجل (دخول → استراحة → إنهاء استراحة → خروج)

### 7. 📍 نظام تتبع الموقع الجغرافي (Geolocation Tracking)

- تحديد موقع الموظف عند تسجيل الدخول باستخدام Browser Geolocation API
- Reverse Geocoding عبر OpenStreetMap Nominatim للحصول على العنوان التفصيلي
- عرض الموقع الكامل: الشارع → الحارة → الحي → المدينة → المحافظة → الدولة
- شاشة إجبارية لتفعيل الموقع كسياسة مؤسسية (لا يمكن الدخول بدون موقع)
- تخزين الإحداثيات وعنوان IP والمتصفح لكل جلسة

### 8. 🛡️ نظام الأمان والصلاحيات (RBAC Security)

- نظام أدوار وصلاحيات ديناميكي كامل (Role-Based Access Control)
- Global Scopes لعزل بيانات كل فرع تلقائياً
- تتبع جلسات المستخدمين في الوقت الفعلي مع حالة Online/Offline
- مراقبة نشاط المستخدم الحالي (أي صفحة يتصفح الآن)
- سجل تدقيق شامل (Audit Trail) لكل عملية في النظام
- حماية CSRF وتشفير كلمات المرور مع OTP للبريد الإلكتروني

### 9. 📝 نظام الامتحانات (Exam Management)

- إدارة امتحانات حضورية وأونلاين
- ربط الامتحانات بالدبلومات والفروع والمدربين
- إدخال درجات مع احتساب النتيجة النهائية
- تقارير نتائج بالتفصيل

### 10. ✅ إدارة المهام (Task Management)

- إنشاء مهام يومية حسب الفرع والموظف
- متابعة حالة التنفيذ مع تغيير سريع للحالة
- تقارير مهام مع أرشفة تلقائية

### 11. 🏖️ نظام الإجازات (Leave Management)

- طلبات إجازة مع دورة موافقة (طلب → موافقة/رفض)
- ربط الإجازات بسجلات الحضور تلقائياً

### 12. 📦 إدارة الأصول (Asset Management)

- تسجيل الأجهزة والمعدات مع حالة الأصل (جيد/صيانة/خارج الخدمة)
- تصنيف حسب الفرع والفئة مع تتبع تكلفة الشراء

### 13. 🏢 إدارة الفروع والدبلومات

- إدارة فروع متعددة (تركيا، ألمانيا، سوريا)
- دبلومات أونلاين وحضورية مع أكواد ورموز فريدة
- ربط كل دبلومة بفرع مع دعم التكرار (نفس الدبلومة بفروع مختلفة)

### 14. 📈 التقارير المتقدمة (Advanced Reporting)

- تقرير تنفيذي شامل مع مؤشرات أداء
- تقارير مالية: يومية، ربحية، إيرادات حسب الفرع
- تقارير حضور: شهرية، أسبوعية مع تصدير PDF/Excel
- تقارير CRM: مصادر العملاء، معدلات التحويل
- رسوم بيانية تفاعلية

### 15. 🎬 إدارة الميديا (Media Management)

- نموذج عام لطلبات الميديا (بدون تسجيل دخول)
- لوحة إدارة داخلية مع جدولة النشر
- تتبع حالة الطلبات

## 🔧 التفاصيل التقنية المتقدمة

### المعمارية (Architecture)

```
├── app/
│   ├── Http/
│   │   ├── Controllers/        # 30+ Controller منظم بـ namespaces
│   │   ├── Middleware/         # 5 Middleware مخصصة
│   │   │   ├── TrackUserActivity    # تتبع نشاط المستخدم لحظياً
│   │   │   ├── AutoLogout           # إنهاء جلسات غير نشطة
│   │   │   ├── PermissionMiddleware # فحص الصلاحيات
│   │   │   └── AuthorizeByRoute     # تحكم وصول حسب الراوت
│   │   └── Requests/           # Form Request Validation
│   ├── Models/                 # 20+ Model مع Relations & Scopes
│   ├── Services/               # Business Logic Layer
│   └── Traits/                 # Auditable Trait
├── database/migrations/        # 25+ Migration منظمة
├── resources/views/            # Blade Templates (RTL Arabic)
│   ├── layouts/                # Master Layout مع Sidebar ديناميكي
│   ├── admin/                  # Users, Roles, Permissions, Audit
│   ├── attendance/             # Attendance + Calendar + Reports
│   ├── crm/                    # Leads + Followups + Reports
│   └── ...                     # 15+ مجلد views
└── routes/web.php              # 100+ Route منظمة
```

### التقنيات المتقدمة غير التقليدية

| التقنية | الوصف |
|---------|-------|
| **Dynamic RBAC with Global Scopes** | عزل تلقائي لبيانات كل فرع عبر Eloquent Global Scopes — الموظف يرى فقط بيانات فرعه |
| **Real-time Session Tracking** | تتبع جلسات المستخدمين مع مراقبة الصفحة الحالية والنشاط اللحظي |
| **Browser Geolocation + Reverse Geocoding** | تحديد موقع GPS + تحويله لعنوان تفصيلي عبر Nominatim API |
| **Polymorphic Financial Accounts** | حساب مالي واحد يربط بطالب أو عميل عبر Morphable Relations |
| **Audit Trail System** | تسجيل كل عملية CRUD في النظام مع المستخدم والوقت والبيانات القديمة/الجديدة |
| **Smart Break System** | نظام استراحة ذكي مع تراكم الاستراحات وحساب صافي ساعات العمل |
| **Mandatory Geofencing** | شاشة إجبارية لتحديد الموقع كسياسة مؤسسية — لا دخول بدون موقع |
| **Multi-Currency Cashbox** | صناديق مالية متعددة العملات مع أرصدة محسوبة وترحيل حركات |
| **Dynamic Diploma Picker** | نظام اختيار دبلومات مع بحث فوري + فلترة فرع + اختيار رمز |
| **IP-based Fallback Geolocation** | تحديد موقع تقريبي من IP كبديل عند رفض GPS |

## ⚡ التثبيت
